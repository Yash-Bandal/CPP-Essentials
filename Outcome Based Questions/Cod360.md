## What will be the size of the object of the following class?

**Assume that integers take 4 bytes.**

```cpp
#include<iostream>
using namespace std;

class Test {
    static int i;
    int j;
};

int Test::i;

int main() {
    cout << sizeof(Test);
    return 0;
}
```

---

## **Answer:**

The **size of an object of class `Test`** is **4 bytes**.

### **Explanation:**
1. **Class Members Analysis:**
   - `static int i;` â†’ **Static members are not stored inside the object** but in **global/static memory**.
   - `int j;` â†’ **This is a non-static member** and is stored within each object.

2. **Memory Calculation:**
   - `int j;` â†’ **4 bytes** (since an `int` takes 4 bytes).
   - **Static data members do not contribute to the object's size.**

3. **Final Result:**
   ```cpp
   cout << sizeof(Test);  // Output: 4
   ```
   - The **size of an object of `Test` is 4 bytes**.

### **Key Takeaways:**
âœ… **Static members do not contribute to object size.**  
âœ… **Only non-static members affect object size.**  
âœ… **Size of `Test` object = size of `int j` = 4 bytes.** ðŸš€
